# Arbitrary Precision Calculator in C

A C-based calculator that performs arithmetic operations on very large integers exceeding built-in data type limits. Uses a doubly linked list to store digits and supports addition, subtraction, multiplication, division, and modulo operations manually.

---

## Features
- Addition and subtraction with digit-wise traversal and carry/borrow handling
- Multiplication using pen-and-paper method with digit shifts
- Division using long division logic
- Modulo operation
- Handles very large integers beyond standard data types
- Modular design with pointer and dynamic memory management

---

## Technologies Used
- C Programming
- Doubly Linked Lists
- Pointers & Dynamic Memory
- Modular Programming
